Как определить свой размер

Для точного определения размера измерьте:

  • Обхват груди - измерьте по самой выступающей точке груди
  • Обхват талии - измерьте по самому узкому месту
  • Обхват бедер - измерьте по самым выступающим точкам ягодиц

Советы по выбору
  • Измеряйте себя перед каждой покупкой
  • Учитывайте материал (хлопок дает усадку)
  • Проверяйте размерную сетку каждого бренда
  • При сомнениях выбирайте больший размер
